# Power-BI---Superstore-
Objetivo de analizar el rendimiento de ventas, la rentabilidad y el comportamiento de productos a partir del dataset Superstore.

📊 KPIs principales:
• Ventas Totales: $2.3M
• Ganancia Total: $292K
• Margen de Ganancia: 13%
• Promedio de tiquetes: 455

📈 ¿Qué incluye el análisis?
✔ Tendencia de ventas vs mes anterior
✔ Rentabilidad por categoría y subcategoría
✔ Impacto de los descuentos en las ganancias
✔ Desempeño y ranking de productos
✔ Relación entre cantidad vendida y ventas

💡 Insight clave:
Altas ventas no siempre significan mayor rentabilidad. Se identificaron productos con buenos ingresos pero con pérdidas debido a altos niveles de descuento.

🛠️ Habilidades aplicadas:
• Modelado de datos
• Creación de medidas en DAX
• Diseño de dashboards con enfoque empresarial
